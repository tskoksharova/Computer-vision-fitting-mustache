MFitting-mustache (OpenCV.js)

Учебный проект по дисциплине «Компьютерное зрение»: позволяет загрузить фото и “примерить” разные варианты усов (PNG с прозрачностью).

Возможности
- Загрузка фотографии пользователя
- Детекция лица с помощью OpenCV.js и Haar Cascade
- Выбор усов из набора изображений (до 10 PNG)
- Настройка положения и размера усов (Scale / X / Y)
- Скачивание результата в PNG

Стек
- HTML / CSS / JavaScript
- OpenCV.js
- Haar Cascade: `haarcascade_frontalface_default.xml`
